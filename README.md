# 赛博算卦 - AI六爻预测程序

一个精美的苹果风格网页算卦程序，基于六爻预测思想，使用AI技术为您解读卦象。

## 功能特色

🎯 **传统六爻预测**: 根据《易经》六爻占卜原理，生成六个爻位的完整卦象

🤖 **AI智能解读**: 使用kimi-k2-0905-preview（Moonshot AI）模型专业解读卦象

✨ **苹果风格设计**: 现代化UI设计，渐变背景、毛玻璃效果、流畅动画

🎨 **精美图形化展示**: 卦象以精美图形化方式展现，包含详细注释和说明

🔮 **智能卦象分析**: 自动识别上下卦、分析卦象结构、提供详细解读

🗣️ **说人话功能**: 点击按钮可切换通俗易懂版本，让占卜更接地气

📸 **小红书卡片**: 一键生成精美的分享卡片，可保存到本地分享到社交媒体

## 使用方法

### 1. 获取 API Key

由于程序使用 Moonshot AI 的 API，您需要先获取 API Key：

- 访问 [Moonshot AI开放平台](https://platform.moonshot.cn/console/apiKeys)
- 注册/登录账号
- 创建 API Key

### 2. 启动程序

在浏览器中直接打开 `index.html` 文件即可使用程序。

### 3. 首次使用

- 输入 API Key 并保存（将自动保存在浏览器本地存储中）
- 之后可随时点击设置按钮重新设置 API Key

### 4. 开始占卜

1. 在输入框中输入您要占卜的事件
2. 点击「开始摇卦」按钮
3. 系统将随机生成六爻卦象
4. 查看详细的图形化卦象解读和AI解读

### 5. 生成分享卡片 (可选)

1. 占卜完成后，点击左侧或右侧的「生成卡片」按钮
2. 在弹出的模态框中查看生成的精美卡片
3. 点击「下载图片」将卡片保存到本地
4. 分享卡片到小红书、朋友圈等社交平台

**注意**：卡片采用小红书风格设计，包含卦象信息、二维码和日期，非常适合社交分享

## 六爻卦象解释

### 爻位含义（从下到上）
- **初爻**: 此事的开端
- **二爻**: 此事的过程（分内之事）
- **三爻**: 此事的过程（发展变化）
- **四爻**: 此事的发展（外部影响）
- **五爻**: 最终结果（成败关键）
- **上爻**: 最终结果（超出预期）

### 阴阳爻符号
- **阳爻（实线）**: ===爻=== 代表刚健、主动、光明
- **阴爻（断线）**: ==爻== 代表柔顺、内敛、含蓄

## 占卜流程

1. **起心动念**: 专注思考您要占卜的问题
2. **输入事件**: 在输入框中清晰描述问题
3. **摇卦**: 点击按钮生成六爻卦象
4. **观象**: 观察卦象的整体形态
5. **分析**: 查看各个爻位的含义
6. **解读**: 阅读AI提供的详细解读
7. **领悟**: 结合自身情况领悟启示
